DEFINICION DEL PROBLEMA

Un conocido hospital veterinario, está solicitando el desarrollo de una aplicación web para la utilización desu personal y de sus clientes.Desarrolla el diagrama E-R, el diagrama relacional, y el modelo físico, según las siguientes consideraciones: El hospital atiende diferentes tipos de mascotas, perros, gatos, aves, entre otras especies. Cada clientepuede acudir con diferentes especies de mascotas. Los servicios del hospital veterinario pueden ser consulta, urgencia, estética, venta de alimento, venta deaccesorios. En el hospital atienden 3 médicos veterinarios en los diferentes turnos.
